# Memo Command
CLI memo tool


## Usage
```
memo 0.1.0
k-nasa <htilcs1115@gmail.com>
CLI memo command

USAGE:
    memo [SUBCOMMAND]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    config    edit config file
    delete    delete memos
    edit      edit memo
    help      help
    list      show memos list
    new       create new memo
```
